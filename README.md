# Nodepop API - Práctica JS/Node.js/MongoDB - KeepCoding

Backend para dar soporte a una aplicación de venta de artículos de segunda mano.

## Instrucciones

Requisitos de instalación:

- Node
- MongoDB

### Instalación

    $ npm install


### Arrancar servidor

    $ npm start


### Arrancar la base de datos

    $ npm run installDB
